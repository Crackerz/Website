RSSFeed

RSSFeed is a simple open source application for managing RSS feeds. Currently not feature complete, it selling point is its adherence to the Android design principals. It boasts a user friendly, intutitive, and visually appealing user interface and has a highly maintainable code base. Using CardsUI, it offers a simple grouping of article content that is easy to understand and is visually appealing. Built ontop of the Informa Project, it supports RSS 0.9x, RSS 1.0 / RDF, RSS 2.0, Atom 0.3 and Atom 1.0. In addition, it uses jsoup to grab the first occurence of an image and displays it in the article's card. It currently requires altering code to add and remove RSS feeds, but a solution built on ActionBarSherlock and SlidingMenu is in the works.
